Profile Sidebar Widget

There are two things that you should note:

  • If you want to Gravatar, fill in your email in the gravatar field. Otherwise, leave it blank in case it overrides the avatar setting.

  • The social_links field accepts an array of links which are either shown as a text link or icon link. The details are described in Polymorphic Link Settings.
